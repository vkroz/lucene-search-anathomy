boolean org.apache.lucene.search.QueryCachingPolicy.shouldCache ( Query  query) throws IOException

Whether the given Query is worth caching.

This method will be called by the QueryCache to know whether to cache. It will first attempt to load a DocIdSet from the cache. If it is not cached yet and this method returns true then a cache entry will be generated. Otherwise an uncached scorer will be returned.
