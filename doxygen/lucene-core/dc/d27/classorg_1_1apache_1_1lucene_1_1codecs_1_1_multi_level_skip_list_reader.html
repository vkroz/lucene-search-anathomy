Detailed Description

This abstract class reads skip lists with multiple levels.

See MultiLevelSkipListWriter for the information about the encoding of the multi level skip lists.

Subclasses must implement the abstract method readSkipData(int, IndexInput) which defines the actual format of the skip data. .experimental
