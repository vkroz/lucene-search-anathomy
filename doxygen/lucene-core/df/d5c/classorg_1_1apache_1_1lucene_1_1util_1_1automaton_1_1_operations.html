Determinizes the given automaton.

Worst case complexity: exponential in number of states.

Parameters
maxDeterminizedStatesMaximum number of states created when determinizing. Higher numbers allow this operation to consume more memory but allow more complex automatons. Use DEFAULT_MAX_DETERMINIZED_STATES as a decent default if you don't know how many to allow.
Exceptions
TooComplexToDeterminizeExceptionif determinizing a creates an automaton with more than maxDeterminizedStates

static

Returns the topological sort of all states reachable from the initial state.

Behavior is undefined if this automaton has cycles. CPU cost is O(numTransitions), and the implementation is recursive so an automaton matching long strings may exhaust the java stack.
