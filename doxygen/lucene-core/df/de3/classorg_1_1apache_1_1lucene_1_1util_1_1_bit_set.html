int org.apache.lucene.util.BitSet.approximateCardinality ( )

Return an approximation of the cardinality of this set.

Some implementations may trade accuracy for speed if they have the ability to estimate the cardinality of the set without iterating over all the data. The default implementation returns cardinality().
