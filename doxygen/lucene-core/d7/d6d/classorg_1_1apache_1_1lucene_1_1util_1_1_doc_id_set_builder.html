Detailed Description

A builder of DocIdSets.

At first it uses a sparse structure to gather documents, and then upgrades to a non-sparse bit set once enough hits match.

To add documents, you first need to call grow in order to reserve space, and then call BulkAdder#add(int) on the returned BulkAdder.
