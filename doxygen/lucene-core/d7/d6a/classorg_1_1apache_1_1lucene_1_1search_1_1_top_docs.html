Returns a new TopFieldDocs, containing topN results across the provided TopFieldDocs, sorting by the specified Sort.

Each of the TopDocs must have been sorted by the same Sort, and sort field values must have been filled (ie, fillFields=true must be passed to TopFieldCollector#create).
