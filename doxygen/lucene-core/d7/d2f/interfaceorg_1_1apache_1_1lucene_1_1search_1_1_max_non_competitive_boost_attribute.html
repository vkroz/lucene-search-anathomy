Detailed Description

Add this Attribute to a fresh AttributeSource before calling MultiTermQuery#getTermsEnum(Terms,AttributeSource).

FuzzyQuery is using this to control its internal behaviour to only return competitive terms.

Please note: This attribute is intended to be added by the MultiTermQuery.RewriteMethod to an empty AttributeSource that is shared for all segments during query rewrite. This attribute source is passed to all segment enums on MultiTermQuery#getTermsEnum(Terms,AttributeSource). TopTermsRewrite uses this attribute to inform all enums about the current boost, that is not competitive. .internal
