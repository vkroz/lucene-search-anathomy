abstract List<LeafReaderContext> org.apache.lucene.index.IndexReaderContext.leaves ( ) throws UnsupportedOperationException
abstract

Returns the context's leaves if this context is a top-level context.

For convenience, if this is an LeafReaderContext this returns itself as the only leaf.

Note: this is convenience method since leaves can always be obtained by walking the context tree using children().

Exceptions
UnsupportedOperationExceptionif this is not a top-level context.
