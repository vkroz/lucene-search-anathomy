Detailed Description

This class can be used to hold any query configuration and no field configuration.

For field configuration, it creates an empty FieldConfig object and delegate it to field config listeners, these are responsible for setting up all the field configuration.

QueryConfigHandler should be extended by classes that intends to provide configuration to QueryNodeProcessor objects.

The class that extends QueryConfigHandler should also provide FieldConfig objects for each collection field.
