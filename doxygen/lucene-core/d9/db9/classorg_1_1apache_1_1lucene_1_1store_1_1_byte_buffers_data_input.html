Detailed Description

A DataInput implementing RandomAccessInput and reading data from a list of ByteBuffers.

Constructor & Destructor Documentation

◆ ByteBuffersDataInput()

org.apache.lucene.store.ByteBuffersDataInput.ByteBuffersDataInput ( List< ByteBuffer >  buffers)

Read data from a set of contiguous buffers.

All data buffers except for the last one must have an identical remaining number of bytes in the buffer (that is a power of two). The last buffer can be of an arbitrary remaining length.
