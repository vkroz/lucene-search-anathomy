Detailed Description

A Collector which allows running a search with several Collectors.

It offers a static wrap method which accepts a list of collectors and wraps them with MultiCollector, while filtering out the null null ones.

static Collector org.apache.lucene.search.MultiCollector.wrap ( Iterable<? extends Collector >  collectors)
static

Wraps a list of Collectors with a MultiCollector.

This method works as follows:

  • Filters out the null collectors, so they are not used during search time.
  • If the input contains 1 real collector (i.e. non-null ), it is returned.
  • Otherwise the method returns a MultiCollector which wraps the non-null ones.
Exceptions
IllegalArgumentExceptionif either 0 collectors were input, or all collectors are null.
