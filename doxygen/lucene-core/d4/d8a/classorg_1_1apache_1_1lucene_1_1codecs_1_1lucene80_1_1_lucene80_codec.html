Detailed Description

Implements the Lucene 8.0 index format, with configurable per-field postings and docvalues formats.

If you want to reuse functionality of this codec in another codec, extend FilterCodec.

See also
org.apache.lucene.codecs.lucene80 package documentation for file format details.
