Detailed Description

A PagedGrowableWriter.

This class slices data into fixed-size blocks which have independent numbers of bits per value and grow on-demand.

You should use this class instead of the PackedLongValues related ones only when you need random write-access. Otherwise this class will likely be slower and less memory-efficient. .internal
