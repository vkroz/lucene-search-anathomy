Detailed Description

Class to construct DFAs that match a word within some edit distance.

Implements the algorithm described in: Schulz and Mihov: Fast String Correction with Levenshtein Automata .experimental

◆ toAutomaton() [1/2]

Automaton org.apache.lucene.util.automaton.LevenshteinAutomata.toAutomaton ( int  n)

Compute a DFA that accepts all strings within an edit distance of n.

All automata have the following properties:

  • They are deterministic (DFA).
  • There are no transitions to dead states.
  • They are not minimal (some transitions could be combined).
