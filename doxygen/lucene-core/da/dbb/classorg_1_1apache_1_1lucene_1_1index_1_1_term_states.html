Detailed Description

Maintains a IndexReader TermState view over IndexReader instances containing a single term.

The TermStates doesn't track if the given TermState objects are valid, neither if the TermState instances refer to the same terms in the associated readers.

.experimental

static

Creates a TermStates from a top-level IndexReaderContext and the given Term.

This method will lookup the given term in all context's leaf readers and register each of the readers containing the term in the returned TermStates using the leaf reader's ordinal.

Note: the given context must be a top-level context.
