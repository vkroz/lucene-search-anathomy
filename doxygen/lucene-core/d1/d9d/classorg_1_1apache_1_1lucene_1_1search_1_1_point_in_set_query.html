Detailed Description

Abstract query class to find all documents whose single or multi-dimensional point values, previously indexed with e.g.

IntPoint, is contained in the specified set.

This is for subclasses and works on the underlying binary encoding: to create range queries for lucene's standard

Point

types, refer to factory methods on those classes, e.g. IntPoint.newSetQuery() for fields indexed with IntPoint.
