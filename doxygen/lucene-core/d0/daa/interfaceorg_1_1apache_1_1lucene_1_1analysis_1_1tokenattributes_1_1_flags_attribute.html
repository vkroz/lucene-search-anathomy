Detailed Description

This attribute can be used to pass different flags down the Tokenizer chain, e.g.

from one TokenFilter to another one.

This is completely distinct from TypeAttribute, although they do share similar purposes. The flags can be used to encode information about the token for use by other org.apache.lucene.analysis.TokenFilters. .experimental While we think this is here to stay, we may want to change it to be a long.
