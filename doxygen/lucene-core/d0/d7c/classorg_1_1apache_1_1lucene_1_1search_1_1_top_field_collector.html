Detailed Description

A Collector that sorts by SortField using FieldComparators.

See the create(org.apache.lucene.search.Sort, int, int) method for instantiating a TopFieldCollector.

.experimental

Parameters
sortthe sort criteria (SortFields).
numHitsthe number of results to collect.
totalHitsThresholdthe number of docs to count accurately. If the query matches
totalHitsThreshold
hits or more then its hit count will be a lower bound. On the other hand if the query matches less than
totalHitsThreshold
hits then the hit count of the result will be accurate. Integer#MAX_VALUE may be used to make the hit count accurate, but this will also make query processing slower.
Returns
a TopFieldCollector instance which will sort the results by the sort criteria.
