Detailed Description

A visitor that adds every field it sees.

Use like this: 

MergeVisitor visitor = new MergeVisitor(mergeState, readerIndex);
for (...) {
  startDocument();
  storedFieldsReader.visitDocument(docID, visitor);
  finishDocument();
}
