Detailed Description

Estimates the size (memory representation) of Java objects.

This class uses assumptions that were discovered for the Hotspot virtual machine. If you use a non-OpenJDK/Oracle-based JVM, the measurements may be slightly wrong.

static long org.apache.lucene.util.RamUsageEstimator.shallowSizeOf ( Object  obj)
static

Estimates a "shallow" memory usage of the given object.

For arrays, this will be the memory taken by array storage (no subreferences will be followed). For objects, this will be the memory taken by the fields.

JVM object alignments are also applied.

◆ shallowSizeOfInstance()

static long org.apache.lucene.util.RamUsageEstimator.shallowSizeOfInstance ( Class<?>  clazz)
static

Returns the shallow instance size in bytes an instance of the given class would occupy.

This works with all conventional classes and primitive types, but not with arrays (the size then depends on the number of elements and varies from object to object).
