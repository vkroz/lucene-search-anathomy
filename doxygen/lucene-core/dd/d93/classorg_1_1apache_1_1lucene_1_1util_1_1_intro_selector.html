Detailed Description

Implementation of the quick select algorithm.

It uses the median of the first, middle and last values as a pivot and falls back to a heap sort when the number of recursion levels exceeds

2 lg(n)

, as a consequence it runs in linear time on average and in

n log(n)

time in the worst case.
