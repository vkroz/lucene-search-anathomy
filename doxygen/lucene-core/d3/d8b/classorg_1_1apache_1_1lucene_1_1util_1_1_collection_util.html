Detailed Description

Methods for manipulating (sorting) collections.

Sort methods work directly on the supplied lists and don't copy to/from arrays before/after. For medium size collections as used in the Lucene indexer that is much more efficient.

Sorts the given random access List using the Comparator.

The list must implement RandomAccess. This method uses the intro sort algorithm, but falls back to insertion sort for small lists.

Exceptions
IllegalArgumentExceptionif list is e.g. a linked list without random access.
