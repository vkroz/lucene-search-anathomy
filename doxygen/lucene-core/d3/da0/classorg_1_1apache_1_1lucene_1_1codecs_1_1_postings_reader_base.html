Detailed Description

The core terms dictionaries (BlockTermsReader, BlockTreeTermsReader) interact with a single instance of this class to manage creation of org.apache.lucene.index.PostingsEnum and org.apache.lucene.index.PostingsEnum instances.

It provides an IndexInput (termsIn) where this class may read any previously stored data that it had written in its corresponding PostingsWriterBase at indexing time. .experimental
